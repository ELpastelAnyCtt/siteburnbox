# Site Burnbox

Este é um site de apresentação simples, desenvolvido com HTML, CSS e JavaScript.

## Descrição

O site apresenta informações sobre o projeto Burnbox, incluindo funcionalidades, tecnologias utilizadas e como contribuir.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/ELpastelAnyCtt/siteburnbox.git
   ```

2. Abra o arquivo `index.html` no seu navegador.
